# Flexy Box

A simple bloat free flexbox grid library written in SCSS

Work in Progress...

## Feature Roadmap
* Offsets
* Documentation/Examples
* Live Demo


## Supported Browsers
* Chrome
* Firefox
* Safari
* IE 10+
* Opera 12.1+
* iOS
* Android

http://codepen.io/johnnyazee/pen/AXoWkm

## Contributing
You must be running nodejs version => 6.3.0
```
npm install
gulp
```
