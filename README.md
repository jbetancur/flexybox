# Flexy Box

a simple lightweight cross browser flex grid framework written in SCSS

## Live Demo
http://flexybox-docs.pubstorm.site/

## Feature Roadmap
* Offsets

## Supported Browsers
* Chrome
* Firefox
* Safari
* Edge
* IE 10 & 11
* Opera 12.1+
* iOS
* Android

## CodePen
http://codepen.io/johnnyazee/pen/AXoWkm

## Development
You must be running nodejs version => 6.3.0
```
npm install
gulp
```
